## Q2
加载一个X元素需要4B，加载一个Y元素也需要4B，存储一个结果需要4B，在写结果时由于不会直接内内存，CPU实际上是先写cache，然后刷新一个cacheline到内存，因此每次写入操作会涉及到一个cacheline的读取和写入，因此每个结果有额外一个读，因此总共是`4 * N * sizeof(float)`。
```c
const unsigned int TOTAL_BYTES = 4 * N * sizeof(float);
```
